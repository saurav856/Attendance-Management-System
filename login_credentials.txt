INSTALLATION INSTRUCTIONS

1. Install XAMPP (with PHP 8.0.9 or later)
2. Start Apache and MySQL services
3. Open phpMyAdmin (http://localhost/phpmyadmin)
4. Import the db.sql file to create database and tables
5. Place the project folder in htdocs directory
6. Access the application at http://localhost/25123840/
